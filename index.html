Repaso Teórico

Variables y Tipos de Datos

Los 4 tipos de datos fundamentales que hemos trabajado hasta ahora.

# ── TIPOS DE DATOS ──
edad = 25                    # int (entero)
precio = 19.99                # float (decimal)
nombre = "Carlos"              # str (texto)
es_estudiante = True         # bool (verdadero/falso)

# ── CONVERSIÓN DE TIPOS ──
entrada = input("Tu edad: ")  # entrada es str
edad = int(entrada)            # convertir a int
altura = float(input("Altura: ")) # directamente a float
💡

Recuerda: input() siempre devuelve un string. Si necesitas operar con números, debes convertir con int() o float().

Repaso Teórico

Los 7 Operadores Aritméticos

Todas las operaciones matemáticas que necesitamos para la calculadora.

a = 17
b = 5

# ── OPERADORES BÁSICOS ──
suma      = a + b   # 22
resta     = a - b   # 12
producto  = a * b   # 85
division  = a / b   # 3.4 (siempre float)

# ── OPERADORES ESPECIALES ──
div_entera = a // b  # 3 (parte entera)
modulo     = a % b   # 2 (residuo)
potencia   = a ** b  # 1419857 (17 elevado a la 5)
⚠️

Cuidado con la división: / siempre devuelve float, incluso si el resultado es entero. // descarta decimales.

05 / 17
Repaso Teórico

Formateo con f-strings

La forma moderna y profesional de mostrar resultados en Python.

nombre = "Ana"
edad = 23
promedio = 4.567891

# ── INTERPOLACIÓN BÁSICA ──
print(f"Hola, {nombre}. Tienes {edad} años.")
# → Hola, Ana. Tienes 23 años.

# ── FORMATEO DE DECIMALES ──
print(f"Promedio: {promedio:.2f}")
# → Promedio: 4.57

# ── EXPRESIONES DENTRO DE f-strings ──
print(f"El doble de tu edad es {edad * 2}")
# → El doble de tu edad es 46
💡

Sintaxis: :.2f significa "mostrar con 2 decimales". Puedes usar .0f, .3f, .4f, etc.

Debugging Intensivo

NameError — Variable No Definida

El error más común: intentar usar una variable que no existe.

❌ ERROR 
print(resultado)
resultado = 10 + 20

# NameError: name 'resultado'
# is not defined
✅ CORRECTO 
resultado = 10 + 20
print(resultado)

# → 30
🔍

Diagnóstico: Python lee el código de arriba hacia abajo. Una variable debe existir antes de usarla.

07 / 17
Debugging Intensivo

TypeError — Operación Inválida

Intentar operar con tipos incompatibles.

❌ ERROR 
edad = input("Edad: ")
doble = edad * 2
print(doble)

# Si escribes "25":
# → 2525 (repite el string)
✅ CORRECTO 
edad = int(input("Edad: "))
doble = edad * 2
print(doble)

# Si escribes "25":
# → 50
🔍

Diagnóstico: input() siempre devuelve texto. Para hacer matemáticas, convierte a int() o float().

08 / 17
Debugging Intensivo

ValueError — Conversión Imposible

Intentar convertir texto que no es un número válido.

❌ ERROR 
entrada = "veinte"
numero = int(entrada)

# ValueError: invalid literal
# for int() with base 10: 'veinte'
✅ CORRECTO 
entrada = "20"
numero = int(entrada)
print(numero)

# → 20
🔍

Diagnóstico: int() solo acepta texto que represente un número válido: "123" ✅, "doce" ❌, "12.5" ❌ (usa float())

PEP 8 — Estilo Profesional

Las reglas de estilo que hacen tu código legible y profesional.

❌ MAL ESTILO 
NombreCompleto=input("nombre: ")
EDAD=int(input("edad: "))
r=EDAD*365
print("dias vividos:",r)
✅ PEP 8 CORRECTO 
# Variables en snake_case
nombre_completo = input("Nombre: ")
edad = int(input("Edad: "))
dias_vividos = edad * 365
print(f"Días vividos: {dias_vividos}")
  • ✅ Variables en snake_case (minúsculas con guiones bajos)
  • ✅ Constantes en MAYÚSCULAS
  • ✅ Espacios alrededor de operadores: a = b + c
  • ✅ Nombres descriptivos: total_precio mejor que t
  • ✅ Comentarios útiles con #

    La calculadora más simple: pedir dos números y calcular las 7 operaciones.

    INSTRUCCIÓN

    Crear archivo calculadora_v1.py con este código base: 

    # ── CALCULADORA V1 — BÁSICA ──
# Entrada de datos
num1 = float(input("Primer número: "))
num2 = float(input("Segundo número: "))

# Cálculos
suma = num1 + num2
resta = num1 - num2
producto = num1 * num2
division = num1 / num2
div_entera = num1 // num2
modulo = num1 % num2
potencia = num1 ** num2

# Salida de resultados
print(f"\nRESULTADOS:")
print(f"Suma: {suma:.2f}")
print(f"Resta: {resta:.2f}")
print(f"Producto: {producto:.2f}")
print(f"División: {division:.2f}")
print(f"División entera: {div_entera:.2f}")
print(f"Módulo: {modulo:.2f}")
print(f"Potencia: {potencia:.2f}")
    ✅

    Prueba tu código

    Versión 2 — Selección de Operación

    Ahora el usuario elige qué operación quiere ver.

    👉

    MEJORA

    Crear calculadora_v2.py — copia V1 y agrega menú + diccionario: 

    # ── CALCULADORA V2 — CON MENÚ ──
# [... código de V1 hasta las operaciones ...]

# Menú de selección
print("\n¿Qué operación quieres ver?")
print("1. Suma")
print("2. Resta")
print("3. Producto")
print("4. División")
print("5. División entera")
print("6. Módulo")
print("7. Potencia")

opcion = int(input("\nOpción (1-7): "))

# Diccionario de resultados
resultados = {
    1: ("Suma", suma),
    2: ("Resta", resta),
    3: ("Producto", producto),
    4: ("División", division),
    5: ("División entera", div_entera),
    6: ("Módulo", modulo),
    7: ("Potencia", potencia)
}

nombre_op, valor = resultados[opcion]
print(f"\n{nombre_op}: {valor:.2f}")
    💡

    Concepto nuevo: Los diccionarios permiten asociar números (claves) con valores. Aquí usamos tuplas (nombre, resultado) para guardar ambos datos.

    EXPANSIÓN

    Crear calculadora_v3.py — copia V2 y agrega esta sección antes del menú: 

    # ── EXTRAS Y ESTADÍSTICAS ──
promedio = (num1 + num2) / 2
mayor = max(num1, num2)
menor = min(num1, num2)
absoluto_diff = abs(num1 - num2)

# Conversiones (suponiendo que num1 es temperatura)
celsius_a_fahrenheit = (num1 * 9/5) + 32
metros_a_pies = num1 * 3.28084

print(f"\n📊 ESTADÍSTICAS:")
print(f"Promedio: {promedio:.2f}")
print(f"Mayor: {mayor:.2f}")
print(f"Menor: {menor:.2f}")
print(f"Diferencia absoluta: {absoluto_diff:.2f}")

print(f"\n🔄 CONVERSIONES (usando num1={num1}):")
print(f"{num1:.1f}°C = {celsius_a_fahrenheit:.1f}°F")
print(f"{num1:.2f} metros = {metros_a_pies:.2f} pies")

    Funciones útiles: max(), min(), abs() son funciones incorporadas en Python que simplifican cálculos comunes.
